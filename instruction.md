## English

### About RollOrNot

RollOrNot is an AI-powered study companion that helps students learn smarter. Simply snap a photo of any homework or exam question, and get instant AI-powered analysis with detailed explanations, step-by-step solutions, and personalized practice questions.

### AI Model Configuration

RollOrNot supports both **local AI models** and **custom AI providers**. To use AI features, you must configure at least one AI model.

**Local AI models** (Ollama, LM Studio, llama.cpp):
1. Install and run a local AI service on your device or local network
2. Open the app's **Settings** → **AI Model Configuration**
3. Tap **Add** → **Local Model**, then enter the base URL and model name

**Custom AI providers** (e.g., OpenAI, Claude, DeepSeek):
1. Open the app's **Settings** → **AI Model Configuration**
2. Tap **Add** → **Custom Provider**
3. Enter the API base URL, model name, and your API key

> **Note**: Local models keep all data on your device. Custom providers require an active internet connection and a valid API key.

### Content Permit Notice

Chinese law requires apps with news, religious, book, and magazine content to have relevant permits in order to be available on the App Store in China mainland. If applicable, enter information about your permits below.

RollOrNot is an AI-powered study tool designed for educational purposes. It does not provide news, religious, book, or magazine content.

## 简体中文

### 关于 RollOrNot

RollOrNot（卷不卷）是一款 AI 智能学习助手，帮助学生更聪明地学习。只需拍下作业或试题照片，即可获得 AI 驱动的详细解析、逐步解题思路和个性化练习题目。

### AI 模型配置

RollOrNot 同时支持**本地 AI 大模型**和**自定义 AI 服务商**。要使用 AI 功能，您需要配置至少一个 AI 模型。

**本地 AI 大模型**（Ollama、LM Studio、llama.cpp）：
1. 在设备或本地网络中安装并运行本地 AI 服务
2. 在 App 的 **设置** → **AI 模型配置** 中点击 **添加** → **本地模型**
3. 输入本地模型的地址和模型名称

**自定义 AI 服务商**（如 OpenAI、Claude、DeepSeek 等）：
1. 在 App 的 **设置** → **AI 模型配置** 中点击 **添加** → **自定义服务商**
2. 输入服务商的 API 地址、模型名称及 API Key

> **注意**：使用本地模型时，数据留在设备本地；使用自定义服务商时，需要网络连接和有效的 API Key。

### 内容许可说明

根据中国法律，包含新闻、宗教、图书和杂志类内容的应用程序需要取得相关许可，方可在中国大陆的 App Store 上架。如适用，请在下方填写您的许可信息。

RollOrNot 是一款 AI 驱动的学习工具，仅用于教育目的，不提供新闻、宗教、图书或杂志内容。
