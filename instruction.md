## English

### About RollOrNot

RollOrNot is an AI-powered study companion that helps students learn smarter. Simply snap a photo of any homework or exam question, and get instant AI-powered analysis with detailed explanations, step-by-step solutions, and personalized practice questions.

### Bring Your Own AI API Key

RollOrNot does **not** provide built-in AI model quotas. To use AI features, you must:

1. Register with a supported AI provider (OpenAI, DeepSeek, Google Gemini, Claude, etc.)
2. Obtain your own API key from that provider
3. Configure the API key in the app's **Settings** → **AI Model Configuration**

Supported providers include cloud services (OpenAI, DeepSeek, Google Gemini, Claude, Qwen, Doubao, ZhipuAI, GitHub Copilot) and local models (Ollama, LM Studio, llama.cpp).

### Content Permit Notice

Chinese law requires apps with news, religious, book, and magazine content to have relevant permits in order to be available on the App Store in China mainland. If applicable, enter information about your permits below.

RollOrNot is an AI-powered study tool designed for educational purposes. It does not provide news, religious, book, or magazine content.

## 简体中文

### 关于 RollOrNot

RollOrNot（卷不卷）是一款 AI 智能学习助手，帮助学生更聪明地学习。只需拍下作业或试题照片，即可获得 AI 驱动的详细解析、逐步解题思路和个性化练习题目。

### 自带 AI API 密钥

RollOrNot **不提供**内置 AI 模型额度。要使用 AI 功能，您需要：

1. 在支持的 AI 提供商（OpenAI、DeepSeek、Google Gemini、Claude 等）注册

3. 在 App 的 **设置** → **AI 模型配置** 中配置 API 密钥

支持的提供商包括云服务（OpenAI、DeepSeek、Google Gemini、Claude、通义千问、豆包、智谱清言、GitHub Copilot）和本地模型（Ollama、LM Studio、llama.cpp）。
