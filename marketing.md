### Transform the Way You Study

RollOrNot puts the power of AI in your hands. Simply snap a photo of any homework or exam question, and get instant AI-powered analysis with detailed explanations, step-by-step solutions, and personalized practice questions.

---

### Key Features

#### 📸 Smart Question Capture
- Photograph homework, exams, or textbook questions
- Import from photo library or paste text directly
- Supports handwriting, printed text, math formulas, and charts
- Auto-detects subject (Math, Physics, Chemistry, Biology, English, History, Geography, and more)
- Multi-image mode: capture up to 5 images at once with parallel OCR

#### 🤖 AI-Powered Analysis
- Get step-by-step solutions and detailed explanations
- Generate similar practice questions for reinforcement
- AI deep analysis with core knowledge points and common mistake breakdown
- Variant question generation — test the same topic from different angles
- Supports multiple AI providers, including local models (Ollama, LM Studio, llama.cpp)
- Fully configurable AI temperature and timeout settings

#### 📚 Question Bank & Knowledge Management
- Save and organize all analyzed questions
- Full-text search across your entire library
- Create chapters and study books from your questions
- Add personal notes and highlight key concepts
- AI auto-extracts knowledge points from each question
- Generate interactive mind maps from chapter knowledge points
- Export mind maps as PNG, SVG, or PDF

#### 🎮 Interactive Learning Games
- AI dynamically generates HTML5 interactive learning games from chapter key points
- Multiple game types: Word Fill, Math Quiz, Memory Match, Cannon Shooter, Cake Slingshot
- Photo-to-Game: photograph or import images → OCR → AI generates learning games
- Scene animations that visually explain abstract concepts
- Applies Bloom's Taxonomy + spaced repetition for optimal learning

#### 🔍 AR Object Scanning (LiDAR)
- 3D scanning with LiDAR sensor support
- Real-time wireframe mesh visualization with quality color coding
- 360° guided scan for comprehensive coverage
- AI analysis of scanned objects with subject identification
- Generate learning games from real-world objects

#### 📊 Real-Time Grading
- Point your camera at homework for instant AI grading
- Continuous real-time analysis mode
- AR paper detection for flat surface scanning
- Detailed feedback with step-by-step corrections

#### 💾 Backup & Restore
- Create encrypted backups with AES-256-GCM security
- Optional password protection
- Restore or merge multiple backups
- All data stored locally on your device

#### 🌐 Multi-Language Support
- English, Simplified Chinese, Traditional Chinese
- Switch languages instantly in Settings — no restart required
